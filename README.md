- 🛒 **購物車功能**

- 📈 **後臺圖表**：視覺化數據呈現（使用 Chart.js）

---

## 🧪 專案使用技術

| 類別         | 技術項目                  |
|--------------|---------------------------|
| 前端框架     | `Vue.js`                  |
| UI 套件      | `Bootstrap v5.2.3`        |
| 動畫效果     | `wow.js v1.3.0`           |
| 提示訊息     | `sweetalert2 v11.10.1`    |
| 圖表視覺化   | `Chart.js v4.4.2`         |
| DOM 操作     | `jQuery v3.7.1`           |
| 後端資料庫   | `MySQL`                   |
